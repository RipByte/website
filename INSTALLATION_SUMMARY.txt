HOW TO USE
==================================================

1. Open in Browser:
   - Open index.html to see the main portfolio
   - Click "Blog Archive" to view all blogs with search
   - Click any blog card to read full article

2. Add New Blog Post:
   - Create folder: /blogs/blog-title/
   - Add index.html with blog content
   - Update /js/blog-search.js with blog info
   - Update /js/script.js with blog link

3. Use Markdown Converter:
   - Include markdown-converter.js in HTML
   - Call: MarkdownConverter.convert(markdownText)
   - Automatically converts markdown to styled HTML

4. Search Blogs:
   - Type in search bar on blogs.html
   - Results filter in real-time by title
   - Shows count of matching posts
